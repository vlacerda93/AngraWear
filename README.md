# AngraWear

AngraWear é um projeto de estudo de uma loja de **moda** praia, inspirado em uma marca fictícia de Angra dos Reis.[web:16][web:19]  
O objetivo é praticar desenvolvimento web e simular uma loja virtual de biquínis, maiôs e acessórios de praia.

## Tecnologias usadas

- HTML, CSS e JavaScript (ajuste se tiver mais coisas)
- (Ex.: React, Node.js, etc.)

## Funcionalidades (planejadas ou implementadas)

- Listagem de produtos de moda praia
- Página de detalhes do produto
- Carrinho de compras (mesmo que seja só simulação)
- Layout responsivo para desktop e mobile[web:15][web:16]

## Status do projeto

- Em desenvolvimento (projeto de estudo)
